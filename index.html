Prepare For The Workshop

It is important for you to get familiar with HTML, JavaScript (ES6), and basic CSS.

You don’t have to be an expert, basic concepts are all you need!

We created a list of resources below to make it easy-peasy. Don’t worry, we’re here to help and you’ll have an opportunity to ask mentors questions prior to the workshop too!

You don’t have to walk through all of these. We provided different resources so you can find which one works best for you! All resources are free of charge and great ways to learn.

KCWiT’s Coding & Cocktails web development curriculum

https://codingandcocktails.kcwomenintech.org/resources

Review the slides and walk through the worksheets from the following sessions:

  • The Glass: HTML
  • The Liquor: JavaScript
  • The Mixer: Interactive
  • The Rocks: Git is helpful, but optional

If you run into any trouble, feel free to post questions on the #codingandcocktails channel in KCWiT’s Slack team!

Mozilla Developer Network Learn web development tutorial

https://developer.mozilla.org/en-US/docs/Learn

Review the listed sections of the first 4 topics:

  • Complete beginners start here! (optional)
  • HTML - Introduction to HTML
  • CSS - Introduction to CSS
  • JavaScript - JavaScript first steps, JavaScript building blocks, Introducing JavaScript objects, Client-side web APIs: Introduction to web APIs

freeCodeCamp

https://learn.freecodecamp.org/

Review the listed sections of Responsive Web Design and JavaScript Algorithms And Data Structures:

  • Basic HTML and HTML5
  • Basic CSS
  • Basic JavaScript
  • ES6

Codecademy

https://www.codecademy.com/
  • Introduction to HTML (https://www.codecademy.com/learn/learn-html)- free topics only
  • Learn CSS (https://www.codecademy.com/learn/learn-css)- free topics only. CSS Grid section optional
  • Learn JavaScript (https://www.codecademy.com/learn/learn-javascript)- free topics only
